dsFileViewer
directive in module CoreModule

Description

Usage

as element:
<ds-file-viewer
       file-list="{Array}"
       file-name="{String}"
       editable="{Boolean}"
       file-added="{Function}"
       file-removed="{Function}">
</ds-file-viewer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
fileListArray

list of file for display

fileNameString

A name of file for display

editableBoolean

if true you can add or remove file

fileAddedFunction

of file added event

fileRemovedFunction

of file removed event
