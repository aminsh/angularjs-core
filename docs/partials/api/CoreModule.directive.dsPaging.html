dsPaging
directive in module CoreModule

Description

Usage

as element:

       set-page="{Function}"
       total="{Number}"
       options="{Object}">
</ds-paging>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
setPageFunction

directly set page

totalNumber

total items on a request

optionsObject

{pageSize: Number, pagingEnabled: Boolean, reset: Function, refresh: Function}
